# 19.1 `c#14` les nouveautés

## Assignation conditionnel  `nullable` : `null-conditional assignment`

Si on voulait assigner une propriété d'une instance potentiellement `null`, on de vait faire ça :
```cs
Dog? dougy = null;

if(dougy is not null) dougy.Surname = "Dougynou";
```

Maintenant on peu faire cela :

```cs
Dog? dougy = null;

dougy?.Surname = "Dougynou";
```



## le mot clé `field` (version définitive)

```cs
public class Dog
{
    public string? Surname
    {
        get => field?.ToUpper();
        set => field = $"Mister {value}";
    }
}
```

Plus besoin de créer un champ privé explicitement.
